# VS Code Extensions

## Cross Platform (Windows, Linux/WSL)
- Astro - language support, syntax highlighting for .astro files
- Atom One Dark Theme - dark theme based on Atom
- CSS Var Complete - intellisense for CSS variables
- Prettier - code formatter
- Live Server - local dev server with live reload
- Python + Pylance + Python Debugger + Python Environments
- GitHub Repositories - browse and edit GitHub repos without cloning
- GitHub Actions - view and manage workflow runs inside VS Code
- Azure Repos - browse and edit Azure DevOps repos
- Remote Repositories - browse remote repos without cloning
- ESLint - catches JS/TS errors inline
- GitLens - git blame, history, and authorship inline
- Error Lens - inline error and warning highlighting
- Auto Rename Tag - auto renames closing tag when editing opening tag
- Path Intellisense - autocompletes file paths in imports
- Color Highlight - shows color swatches for hex/rgb values in any file
- Git Graph - visual interactive commit history and branch structure
- Thunder Client - lightweight API tester, Postman inside VS Code
- Better Comments - color-coded comment types (warning, todo, important)
- CSS Peek - peek/jump to CSS class definitions from HTML
- vscode-icons - file type icons for better visual navigation
- Code Spell Checker - catches typos in code and comments

## Windows Only
- WSL - required to open WSL folders from Windows

## Notes
- Extensions still must be installed manually on each VS Code environment (WSL, Windows, etc)
- Settings Sync helps but does not always sync to WSL remote automatically
- Install cross platform extensions on every environment you actively code in
